## ✍️ Getting Started

---

### Introduction

Dyndns is a modular skill to be used in composing [autonomous agents](https://aam.wtf/).  It allows you to add dynamic dns agent.

---

### Installation

To get instructions for installing dynamic dns run

```sh
aam skill skill-dyndns
```
